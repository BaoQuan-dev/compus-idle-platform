# 农大闲置平台 - 碳足迹创新模块

## 一、模块概述

碳足迹模块是「双碳+校园循环经济」特色功能的核心，用于量化闲置交易的环保价值，激励用户参与绿色交易。

```

│                      用户界面层                               │
├─────────────────────────────────────────────────────────────┤
│  首页Banner │ 个人中心看板 │ AI助手页面 │ 分享海报           │
├─────────────────────────────────────────────────────────────┤
│                      业务逻辑层                             │
├─────────────────────────────────────────────────────────────┤
```

## 四、Coze AI联动配置

### 4.1 Coze Bot创建

1. 登录 [扣子平台](https://coze.cn)
2. 创建Bot：碳足迹助手
3. 配置人设：

```
# 人设
你是一位专业的碳足迹顾问，服务于河南农业大学闲置交易平台。

## 能力
- 分析用户的碳减排贡献
- 提供个性化环保建议
- 讲解不同品类闲置的环保价值
- 设计碳积分兑换方案
- 科普双碳目标知识

## 风格
- 专业但不晦涩
- 鼓励用户参与
- 结合校园场景
- 数据可视化表达
```

### 4.2 API接口配置

在Coze Bot设置中获取：

```javascript
// Coze API配置
const COZE_CONFIG = {
    BOT_ID: '你的Bot ID',
    API_KEY: '你的API Key',
    API_URL: 'https://api.coze.cn/v1/chat'
};
```

### 4.3 前端对接代码

在`carbon_ai.html`中替换模拟回复为真实API调用：

```javascript
// Coze AI对话
async function sendToCoze(message) {
    const response = await fetch('https://api.coze.cn/v1/chat', {
        method: 'POST',
        headers: {
            'Authorization': 'Bearer ' + COZE_CONFIG.API_KEY,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            bot_id: COZE_CONFIG.BOT_ID,
            user_id: localStorage.getItem('hnau_current_user'),
            query: message,
            stream: false
        })
    });
    
    const data = await response.json();
    return data.messages[0].content;
}
```

### 4.4 扣子工作流配置

在Coze中创建工作流，处理以下意图：

## 六、创新点说明（可直接用于论文）

### 6.1 技术创新

1. **碳足迹量化模型**
   - 基于CLCD数据库建立农大专属碳排放系数表
   - 覆盖7大品类、50+细分品类
   - 考虑新旧程度、交易频次等动态因素

2. **激励机制设计**
   - 碳积分+等级+勋章三维激励体系
   - 游戏化运营提升用户粘性
   - 数据可视化增强成就感

3. **AI智能交互**
   - Coze平台实现对话式环保科普
   - 个性化建议提升用户体验
   - 自然语言处理降低使用门槛

### 6.2 应用创新

1. **校园场景适配**
   - 教材、实验器材等品类贴近学生需求
   - 学院排行榜增强归属感
   - 绿色校园文化契合办学理念

2. **双碳目标响应**
   - 闲置循环直接减少碳排放
   - 量化指标便于宣传推广
   - 可复制推广至其他高校

### 6.3 社会价值

1. **环保教育**
   - 提升学生环保意识
   - 培养低碳生活习惯
   - 传播绿色发展理念

2. **资源循环**
   - 延长物品使用寿命
