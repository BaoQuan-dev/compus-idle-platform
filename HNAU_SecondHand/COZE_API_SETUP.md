# Coze API Key 获取与配置指南

## 一、为什么需要API Key

您的Bot ID是 `7630670043630010374`，但仅凭Bot ID无法直接调用API，还需要获取API Key进行身份认证。

## 二、获取API Key步骤

### 1. 登录扣子平台
访问 [https://coze.cn](https://coze.cn) 并登录您的账号

### 2. 进入个人设置
点击右上角头像 → 选择「个人设置」

### 3. 获取API Key
- 选择「API令牌」选项卡
- 点击「创建令牌」按钮
- 填写令牌名称（如：农大闲置平台）
- 设置权限（建议勾选所有权限）
- 点击创建，复制生成的API Key

### 4. 安全提示
- API Key只会显示一次，请妥善保存
- 不要将API Key直接写在代码中（已改为从localStorage读取）

---

## 三、配置API Key

获取API Key后，在碳足迹助手页面会自动提示配置：

1. 打开 `carbon_ai.html` 页面
2. 打开浏览器开发者工具（F12）
3. 在控制台执行：

```javascript
localStorage.setItem('hnau_coze_api_key', '您的API Key')
```

或者直接在页面的输入框中输入API Key并保存。

---

## 四、测试AI对话

配置完成后，可以：

1. 刷新 `carbon_ai.html` 页面
2. 状态栏应显示「AI助手已就绪」（绿色）
3. 尝试发送问题或点击快捷按钮
4. 查看AI回复

## 五、常见问题

### Q: API调用失败怎么办？
A: 检查：
1. API Key是否正确配置
2. 网络是否正常
3. Bot是否已发布（必须发布才能通过API调用）

### Q: 如何发布Bot？
A:
1. 进入Bot编辑页面
2. 点击右上角「发布」按钮
3. 填写发布说明
4. 确认发布

### Q: 有调用次数限制吗？
A: Coze免费版有调用限制，建议查看官方文档了解具体配额。
